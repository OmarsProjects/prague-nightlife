Google Maps Platform not set
★ Essential
What it unlocks: The big one. Photorealistic 3D Tiles — actual swoop-into-Notre-Dame visualization, real buildings rendered from photos. Plus Places API for accurate venue data and live "popular times" baseline.
$200/mo free credit — covers ~28k 3D tile loads + plenty of Places calls. You won't pay anything for personal use unless you go viral.
  1. Go to console.cloud.google.com → create a project
  2. Enable APIs: Map Tiles API + Places API (New)
  3. Credentials → Create API Key → Copy
  4. (Optional but smart) restrict the key to your domains in production
→ Sign up at Google Cloud Console

BestTime API not set
High Value
What it unlocks: Real "how busy right now" data for every venue, scraped from Google Popular Times. Replaces our estimated heat scores with actual live aggregated phone density. This is the real magic for your scout.
$5–25/mo Starter for ~50–500 venues, $99/mo Pro tier. Free trial available. Cheap relative to what it does.
  1. Sign up at besttime.app
  2. Choose Starter plan (or trial first)
  3. Dashboard → API Keys → Copy
→ Sign up at BestTime
